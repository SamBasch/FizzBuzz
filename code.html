The Code

                    
                        //get the user input from the page

                        //controller function or entry point function (kicks off the whole process)
                        function fetchUserInput() {
                            //get values from page
                        
                        
                            let startValue = document.getElementById('startValue').value;
                        
                            let endValue = document.getElementById('endValue').value;
                        
                            let fizz = document.getElementById('fizzValue').value;
                        
                            let buzz = document.getElementById('buzzValue').value;
                        
                        
                        
                        
                            //parse our inputs as numbers
                        
                            startValue = parseInt(startValue);
                            endValue = parseInt(endValue);
                            fizz = parseInt(fizz);
                            buzz = parseInt(buzz);
                        
                        
                        
                            //verify our inputs are actually numbers (validation)
                        
                            if (Number.isInteger(startValue) && Number.isInteger(endValue) && Number.isInteger(fizz) && Number.isInteger(buzz)) {
                        
                                let numbersArray = generateNumbers(startValue, endValue);
                        
                                displayNumbers(numbersArray, fizz, buzz);
                        
                        
                            } else {
                        
                                Swal.fire(
                                    {
                                        icon: 'error',
                                        title: 'Oops!',
                                        text: 'Only integers are allowed for FizzBuzz!'
                        
                                    }
                                );
                        
                        
                            }
                        
                        
                            //after validation generate our numbrs
                            //display them on page
                        
                        
                        
                        }
                        
                        
                        
                        
                        
                        //generate our numbers
                        
                        function generateNumbers(start, end) {
                        
                            let numbers = [];
                        
                            for (let i = start; i <= end; i++) {
                        
                                numbers.push(i);
                            }
                        
                            return numbers;
                        
                        
                        
                        }
                        
                        
                        
                        //display them on page
                        
                        //view funciton (function is dedicated to making it 'show up' on our web page)
                        function displayNumbers(numbersArray, fizz, buzz) {
                            let tableBody = document.getElementById('results');
                        
                        
                        
                            let tableHtml = "";
                        
                            for (let i = 0; i < numbersArray.length; i++) {
                        
                                let value = numbersArray[i];
                                let className = '';
                        
                                if (value % fizz == 0 && value % buzz == 0) {
                                    className = 'fizzbuzz'
                                    value = 'FIZZBUZZ'
                                    
                                } else if (value % fizz == 0) {
                                    className = 'fizz'
                                    value = "FIZZ"
                                } else if (value % buzz == 0) {
                                    className= 'buzz'
                                    value = 'BUZZ'
                                } else {
                                    
                                }
                        
                        
                                if (i % 5 == 0) {
                                    tableHtml = tableHtml + ''
                                }
                        
                                let tableRow = `${value}`
                        
                                tableHtml = tableHtml + tableRow;
                        
                                if (i + 1 % 5 == 0) {
                                    tableHtml = tableHtml + ''
                                }
                        
                            }
                        
                            tableBody.innerHTML = tableHtml;
                        
                        }

The code is structured in 3 functions

fetchUserInput()

This function takes in the user inputs that set that value for fizz, buzz, as well as the startind and ending value of the number range they would like to check through. This function then parses are the values input from the user by using 'parseInt()'. The parsed numbers are then assigned to a variable with their respective name.

The function then performs a validation check on the parsed inputs to make sure they were able to be parsed as integers. This is down via an 'if statement' If any of the inputs dont pass the validation check you will be greeted with a sweet alert. If the validation check passes the program continues by calling the 'generateNumbers()' function.

generateNumbers()

This function simply creates and an array of numbers. The range/size/length of the array is determined by the 'starting and ending value' of the users input the array is then retruned to the line that called the this function back in the controller function.

displayNumbers()

This function is passes the parameters of the generated numbers array, and the user input value for 'fizz' and 'buzz' using a for loop the array of numbers is iterated through and with every iteration the current number of the array is check with modulo to see if it divisible by the 'fizz' or 'buzz' value. Via and if else/if statement...the current number is FIRST checked to see if it is divisible by BOTH the 'fizz' and 'buzz' values. (this has to be done first because if it is found to by divisible by just 'fizz' or 'buzz' first the if statement would end and you would never be able to see if it was divisible by both). whether the number is divisible both or just 'fizz' or just 'buzz' it will be given the respective class name that will change its appearance via CSS. More importantly its value will be assigned to the string 'fizzbuzz', 'fizz', or 'buzz' so that the string template literal displays it as those words instead of the actual number.
